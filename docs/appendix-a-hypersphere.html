A.1 Algorithm Description

The hypersphere point placement scheme generates \(m\) sample points distributed approximately uniformly on the surface of an \(n\)-dimensional unit hypersphere, then maps them into the interior of the unit hypercube. The algorithm exploits the well-known result that normalised Gaussian random vectors are uniformly distributed on the unit sphere.

A.1.1 Step-by-Step Algorithm

  1. Generate \(m\) vectors \(\mathbf{u}_i \in \mathbb{R}^n\), each drawn from an \(n\)-dimensional standard Gaussian distribution: \[\mathbf{u}_i \sim \mathcal{N}(0, I_n)\]

  2. Normalise each vector to unit length (project to sphere surface): \[\hat{\mathbf{u}}_i = \frac{\mathbf{u}_i}{\|\mathbf{u}_i\|}\]

  3. Map from \([-1, 1]^n\) (sphere surface coordinates) to \([0, 1]^n\): \[\mathbf{x}_i = \frac{\hat{\mathbf{u}}_i + \mathbf{1}}{2}\]

A.1.2 Anti-Bunching Property

For large \(n\), the hypercube has most of its volume concentrated in the corners. Uniform random points in a hypercube tend to cluster in corners, leaving the interior sparsely sampled. The hypersphere scheme avoids this by ensuring points lie on a surface of equal radial distance from the origin — a strong anti-bunching mechanism that produces more even coverage of the design space.

A.2 Minimum Point Count

A key advantage of the hypersphere scheme is that the number of points is determined solely by the minimum required for the RS model (\(N_S\) from Equation 5.9), not by the formula of any specific classical design. This gives the designer freedom to generate exactly as many points as needed.

A.3 Julia Implementation (Full Detail)

function hypersphere_points_full(n::Int, m::Int; seed::Int=42) :: Matrix{Float64}
    # Step 1: n-dimensional Gaussian samples
    rng  = MersenneTwister(seed)
    U    = randn(rng, m, n)             # m × n matrix of raw Gaussian samples

    # Step 2: normalise each row to unit Euclidean norm
    norms = sqrt.(sum(U .^ 2, dims=2))  # m × 1 column of norms
    U_hat = U ./ norms                  # each row now has norm = 1

    # Step 3: map from [-1,1]ⁿ to [0,1]ⁿ
    X = (U_hat .+ 1.0) ./ 2.0

    return X
end
