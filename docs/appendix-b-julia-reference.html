B.1 Flutter Constraint Derivation

The anti-flutter constraint added to the upgraded structures black box is based on a first-order aeroelastic stability criterion. Define:

  • \(M_{DR}\) — deformed wing root bending moment (from ELAPS deflection solution)
  • \(M_{RR}\) — rigid wing root bending moment (from applied aerodynamic loads)

The bending moments are computed from the spanwise lift distribution \(L(y)\):

\[ M_{root} = \int_0^{b/2} L(y) \cdot y \, \mathrm{d}y \]

For the rigid wing, \(L(y)\) is the prescribed aerodynamic load distribution. For the deformed wing, \(L(y)\) is modified by the twist and bending deformations \(d_A(y)\) and \(d_B(y)\) computed by ELAPS:

\[ M_{DR} = \int_0^{b/2} L(y)\left(1 + \frac{\partial C_L}{\partial \alpha}\frac{d_A(y) - d_B(y)}{c(y)}\right) y \, \mathrm{d}y \]

The flutter constraint requires:

\[ \frac{M_{DR}}{M_{RR}} \leq 1.0 \tag{B.1}\]

When \(M_{DR}/M_{RR} > 1\), the deformed wing generates more lift than the rigid wing, which amplifies the deflection — a divergence/flutter instability.

In ELAPS, the ratio is approximated by:

\[ \frac{M_{DR}}{M_{RR}} \approx 1 + K_{flutter} \cdot AR_w^{1.5} \cdot (d_A - d_B)_{\text{max}} \]

where \(K_{flutter}\) is a calibration constant derived from higher-fidelity aeroelastic computations.

B.5 Reproducing the Results

All figures and tables in this book are generated from the source .qmd files. To reproduce:

# 1. Install pixi (https://pixi.sh)
# 2. Install Julia and package dependencies
pixi run setup

# 3. Render to HTML (default)
pixi run render

# 4. Render to PDF
pixi run render-pdf

# 5. Clear cache and re-render from scratch
pixi run clean && pixi run render

Computation time for a cold render (all code executed from scratch) is approximately 15–30 minutes on an Apple M4, dominated by the BLISS-RS optimization loops in Chapters 7 and 10. Subsequent renders use Quarto’s freeze cache and complete in under 1 minute.
