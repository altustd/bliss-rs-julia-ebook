The MDO Challenge

Designing complex systems like supersonic aircraft requires optimizing across multiple disciplines:

  • Structures — Wing weight, aeroelastic twist
  • Aerodynamics — Lift, drag, L/D ratio
  • Propulsion — Engine sizing, fuel consumption
  • Performance — Range, payload, mission capability

Each discipline has its own optimization problem, but they're tightly coupled.

Bi-Level Decomposition

System Level (Outer Loop)

Optimizes global variables: wing geometry, altitude, Mach number

Sends Y* coupling targets →

Disciplinary Level (Inner Loop)

Each black box optimizes its local variables, receives Y* targets

← Returns Y actual outputs

BLISS converges when Y* targets match Y actual outputs across all disciplines

Why Quadratic Polynomials Aren't Enough

◼ Polynomial RS: Misses local maximum, predicts negative range (impossible)

◼ GP Surrogate: Interpolates exactly, provides uncertainty bands where data is sparse

Gaussian Process Surrogates

A GP is a probability distribution over functions:

f(x₁), ..., f(xₙ) ~ 𝒩(m, K) K_ij = k(x_i, x_j) = σ² · exp(-||x_i - x_j||² / (2ℓ²))

Interpolating

Passes exactly through training points

Probabilistic

Provides uncertainty σ(x) everywhere

Adaptive

Hyperparameters ℓ, σ² learned from data

Expected Improvement Acquisition

Where should we evaluate the expensive black box next?

EI balances exploitation (high mean μ) vs exploration (high uncertainty σ)

Key Results

  • GP surrogates achieve 35% lower RMSE than polynomial RS
  • Bayesian optimization converges in 30% fewer iterations
  • Uncertainty estimates enable risk-aware design
  • Monolithic optimizer validation confirms BLISS finds true optimum
  • Parallel evaluation reduces wall-clock time by ~4×

When to Use GP Surrogates

Expensive BBs

CFD, FEA, high-fidelity simulations where every evaluation counts

Noisy Data

GP noise model σ_n² handles stochastic simulations gracefully

Unknown Landscape

When response surface structure is unknown a priori

Polynomial RS still preferred for: low-dimensional, smooth, fast BBs with known structure

Future Directions

  • High-fidelity black boxes (CFD, FEA) replacing empirical models
  • Multi-fidelity GP co-kriging (blend cheap + expensive analyses)
  • Neural network surrogates for very high dimensions
  • Gradient information from adjoint solvers
  • Benchmark against modern MDO frameworks (OpenMDAO)
  • Additional test cases beyond supersonic BJ

Conclusion

BLISS-2026 brings a proven decomposition framework into the modern era with Gaussian Process surrogates, Bayesian optimization, and parallel computing.

The supersonic business jet results demonstrate significant improvements over polynomial response surfaces for complex, coupled MDO problems.
